Что такое Webalizer?

Webilizer — программа для обработки log-файлов и предоставления статистики в виде отчетов. Результаты отображаются в виде графиков и таблиц, облегчающих анализ и интерпретацию статистики посещений вашего сайта.

Какие отчеты генерирует Webalizer?

В первой таблице содержится общая статистика за месяц:

  • Total Hits — Всего запросов — считаются запросы не только html-страниц, но и графических файлов, скриптов и т.д.
  • Total Files — Всего файлов
  • Total Pages — Всего страниц
  • Total Visits — Всего посещений
  • Total KBytes — Общее количество килобайт, переданных сервером.
  • Total Unique Sites — Общее число уникальных IP-адресов, которые делали запросы на сервер.
  • Total Unique URLs — Всего уникальных URL
  • Total Unique Referrers — Всего уникальных ссылающихся страниц
  • Total Unique User Agents — Всего уникальных броузеров

Следующая таблица содержит статистику по кодам возврата сервера:

  • Code 200 — Запрос был успешно обработан
  • Code 206 — Сервер выполнил частичный запрос
  • Code 301 — Ресурс перенесен - запрошенный ресурс сменил свой URI
  • Code 302 — Запрошенный ресурс найден, но временно сменил свой URI
  • Code 304 — Клиент выполнил условный запрос, доступ разрешен, но документ не был изменен
  • Code 400 — Неверный запрос
  • Code 403 — Доступ запрещен
  • Code 404 — Ресурс не найден
  • Code 405 — Данный тип запроса не применим к ресурсу, заданному в URI запроса

Подробнее обо всех возможных кодах возврата вы можете узнать в спецификации HTTP/1.1 на сайте W3C.
